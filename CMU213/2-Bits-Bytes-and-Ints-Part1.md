# Everything is bits
- 为什么要使用bits-- Electronic Implementation：可以通过bit对模拟信号进行量化，将某个低电压范围区间的值用0表示，高电压区间的值用1表示，只要电压在设定的范围内，就能得到一个清晰的信号（数字信号对比模拟信号的主要优势）。
![[Pasted image 20220410224237.png]]
对于信息存储来讲，使用bit这种数字存储信息的方式要比存储一个模拟值要容量的多。
